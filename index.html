URL Shortener Microservice

My implementation of FreeCodeCamp's URL Shortener Microservice API project.

How It Works

  • To create a new 'short-form' URL, pass the original URL as a parameter. This website will create and return a short-form URL in the JSON response.
  • When you navigate to a short-form URL, this website will redirect your browser to your original link.

Example

  • To create a short-form URL for the website www.freecodecamp.com, navigate to https://fcc-url-shortener-agathalynn.c9users.io/http://www.freecodecamp.com.
